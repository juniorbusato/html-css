Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (Não semântica)

Nesta frase temos um termo em negrito usando a tag <strong> (Semântica)

Italico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (Não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Podemos mudar a cor do texto marcado usando no inicio da tag <mark> por exemplo <mark style="background-color: lime;"> - (Isso já é CSS)

E no outro parágrafo, temos outro texto marcado no final

A cor padrão do marcador pode ser definida de forma universal na pagina utilizando o CSS lá em HEAD e abaixo de TITLE, por exemplo:

<style>

mark {

background-color: yellow;

}

</style>

Texto Grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo com as tags <big> e <small>

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Usamos a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usamos a tag <ins>

Existe também o sublinhado com a tag <u> (Não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 Usamos a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O Usamos a tag <sub>
